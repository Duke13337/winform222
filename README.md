# Система учёта заявок «РегионСнаб»

Настольное приложение для автоматизации процесса закупок в офисах и на малых производствах. Проект выполнен в рамках учебной практики.

## 🛠 Технологический стек
*   **Язык:** C# 
*   **Платформа:** .NET 8.0 (Windows Forms)
*   **Хранение данных:** JSON (локальный файл `orders.json`)
*   **Библиотеки:** Newtonsoft.Json (для работы с данными)

## 🚀 Функционал
1.  **Учёт заявок:** Создание, хранение и отображение списка заявок.
2.  **Валидация:** Защита от некорректного ввода (запрет цифр в ФИО, запрет букв в цене).
3.  **Статусы:** Возможность изменения статуса заявки с цветовой индикацией строки (Новая, Заказана, Получена и т.д.).
4.  **Поиск:** Мгновенная фильтрация списка по наименованию товара.
5.  **Аналитика:** Автоматический подсчёт суммы по всем заказанным позициям в заголовке окна.
6.  **Экспорт:** Выгрузка актуального списка в формат CSV для работы в Excel.
7.  **Удаление:** Возможность удаления записей по клавише `Delete` с подтверждением.

## 📦 Инструкция по запуску

### Требования
*   Установленный [.NET SDK 8.0] или выше.

### Запуск приложения
1.  Склонируйте репозиторий или скачайте архив с кодом.
2.  Откройте терминал в папке проекта (`RegionSnabApp`).
3.  Восстановите зависимости (если необходимо):
    ```bash
    dotnet restore
    ```
4.  Запустите приложение командой:
    ```bash
    dotnet run
    ```

## 📂 Структура данных
Данные хранятся в файле `orders.json` в корневой папке приложения. При первом запуске файл создается автоматически. Экспорт отчетов производится в файл `export_orders.csv`.
